# Akash-Vault

A comprehensive Oracle & PostgreSQL DBA knowledge base — queries, commands, procedures, and references, all in one searchable website.

**Live site:** [akashgupta200.github.io/Akash-Vault](https://akashgupta200.github.io/Akash-Vault/)

---

## What's Inside

| Section | Description | Content |
|---------|-------------|---------|
| **Oracle Queries** | Complete Oracle DBA reference | 48 topics — connections, monitoring, sessions, blocking, tuning, backup, user management, tablespaces, Data Guard, Data Pump, and more |
| **PostgreSQL Queries** | Full PostgreSQL DBA guide | 59 topics — architecture, installation, replication (streaming, logical, cascade), backup/restore, PITR, upgrades, pgbench, pgbadger, version features (13–17) |
| **DBA Reference Sheet** | Oracle DBA reference from DBA-Sheet-6.1 | 30 topic areas — OS commands, ASM, RAC, Exadata, performance tuning, RMAN, locks, tracing, OEM, GoldenGate, memory, security, patching |

## Features

- **Dark theme** with SQL syntax highlighting
- **Full-text search** across all content
- **Left sidebar navigation** with expandable sections
- **In-page table of contents** for quick jumping within long pages
- **Browser-based editing** via Decap CMS admin panel — no git or code needed
- **Auto-deploy** on every change via GitHub Actions

## How to Edit Content

1. Go to [akashgupta200.github.io/Akash-Vault/admin/](https://akashgupta200.github.io/Akash-Vault/admin/)
2. Login with your GitHub account
3. Pick the page to edit
4. Use the rich text editor (like Word/Notion)
5. Click **Publish** — site updates in ~60 seconds

## Tech Stack

- [Jekyll](https://jekyllrb.com/) with [Just the Docs](https://just-the-docs.com/) theme
- [Decap CMS](https://decapcms.org/) for browser-based content editing
- GitHub Pages for hosting
- GitHub Actions for CI/CD
